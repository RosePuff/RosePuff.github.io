IntelliJ IDEA 2020.3.1设置

启动后配置

首次启动的时候一般启动的是（/bin/idea64.exe），会有提示说叫我们选择运行的环境，主体UI，勾选上就行了，然后注册自己找破解

正常启动之后如下如图

图中的Setting 既为 在进入idea主界面后的 File → Settings

设置项目默认的JDK配置

==因为在此版本中没有这个选项，需要在进入项目之后再进行选择==

其余版本可以直接选择

  • 进入设置菜单 Project Structure | Project Settings | Project 中选择默认的JDK。（建议选本电脑已经安装的JDK）

==设置后需要点Apply按钮生效==

主题UI以及显示设置

如下图进入设置菜单Settings | Appearance & Behavior | Appearance 选择喜欢的主题。设置后需要点“Apply”按钮生效，这里我用的是插件中的主题（==可能黄色偏多？==）

如图进入菜单中的 Editor | General红色方框，选择根据鼠标滚轮进行放大缩小（本人未启动）

==设置后需要点Apply按钮生效==

进入设置菜单Settings | Editor | General | Editor Tabs，不勾选”Show tabs in one row”\。 这样在打开代码文件较多是，上面的Tab标签可以多行显示，找起来方便一点

同时，勾选 Mark modified(*)当文件被修改后，文件Tab标签带星号，便于识别

==设置后需要点Apply按钮生效==

因为IDEA的默认语法不够醒目，所以进入设置菜单Settings | Editor | Color Scheme | General

  • 在Errors and Warnings下，选择Error，设置Effects波浪线Underwaved，颜色修改为十六进制“FF3F3C”亮红色
  • 在Errors and Warnings下，选择Unknown symbol，去掉勾选Foreground， 设置Effects波浪线Underwaved，颜色修改为“FF3F3C”亮红色
  • 在Code下，分别选择Identifier under caret 和 Identifier under caret(write)，都设置Background颜色十六进制“406000”
  • 在Editor下，选择Selection background，设置Background颜色设置十六进制“406000”

==设置后需要点Apply按钮生效==

设置默认字符编码

如下图，进入设置菜单Settings | Editor | File Encodings，设置默认字符编码（三个地方都设置为UTF-8）

==设置后需要点Apply按钮生效==

设置自动编译

如下图，进入设置菜单Settings | Build, Exceution,Deployment | Compiler；勾选Build project automatically（从eclipse转过来的朋友记得勾选，因为eclipse开发并不用管理编译，源文件落盘的时候自动编译为是最新的，然而IDEA默认不编译，所以需要勾选）

==设置后需要点Apply按钮生效==

设置Auto Import

进入设置菜单Settings | Editor | General | Auto Import，勾选Add unambiguous imports on the fly（自动添加无歧义的import）， 勾选 Optimizeimports on the fly （自动优化，比如删除无用的import）

==设置后需要点Apply按钮生效==

修改默认快捷键

进入设置菜单Settings | Keymap；这里有多种选择，选择一个自己习惯的操作即可

==设置后需要点Apply按钮生效==

代码补全设置

如下图，进入设置菜单Settings | Editor | General | Code Completion

  • 代码补全的时候，匹配候选代码忽略大小写，去掉勾选Match case
  • 代码补全时，勾选Show the documentation popup自动弹出方法用法文档(且等待时间缩短至100毫秒)
  • 代码补全的时候，方法参数信息提示，三个都勾上，可以有类似于eclipse的提示方法参数

==设置后需要点Apply按钮生效==

设置自动添加文件头（用于自动创建信息）又叫模板

如下图，进入设置菜单Settings | Editor | File and Code Templates，直接增加File Header即可

==设置后需要点Apply按钮生效==

隐藏idea自带的.iml文件，以及.idea文件夹

一般来说在文件目录中都是会存在一种idea自带的那个文件以及文件夹，在目录中看着是很不方便，所以避免不必要的干扰的话

进入设置菜单Settings | Editor | File Types。 在界面中找到Ignored File and Folder，老版本在界面最右下角，在Ignored File and Folder列表最后增加“*.iml;*.idea;”——==请注意最后是有个分号的==

==设置后需要点Apply按钮生效==

设置Git

这个Git我们是经常使用的，所以直接配置，因为本人本地已经安装了的那个Git，所以可以说是直接搜索到

对于没有配置的同学来说进入设置菜单Settings | Version Control | Git ，找到Path to Git excutable中输入本地已经安装git.exe全软件的路径

设置Maven

本地电脑已经单独安装了Maven3.6.3，并且在maven config 目录下配置了settings.xml配置文件，所以直接在settings.xml文件中修改

  • 本地存储路径
  • 远程仓库使用的路劲
<!-- 这里大约在五十多行的时候-->
<localRepository>E:\Program_Files\apache-maven-3.6.3\repository</localRepository>

  中间内容省略
 <!--这里大约在一百七十多行的时候-->
<mirror>
  <id>aliyunmaven</id>  
  <mirrorOf>central</mirrorOf>  
  <name>aliyun maven</name>  
  <url>https://maven.aliyun.com/repository/public</url>  
</mirror>

进入设置菜单Settings | Build, Exceution,Deployment | Build Tools | Maven

如下图选择你所需要的版本，因为已经配置好了，不适用IDEA自带的那个maven3的版本了对应下面仓库，以及C盘的settings.xml配置请直接替换

==设置后需要点Apply按钮生效==

设置Gradle

如下图，进入设置菜单Settings | Build, Exceution,Deployment | Build Tools | Gradle设置Home路径即可

关闭自动更新

进入设置菜单Settings | Appearance & Behavior | System Settings | Updates，去掉勾选自动Idea软件更新。 如希望更新，可以手动点 Check Now按钮

安装插件

如下图，进图设置菜单Settings | Plugins，搜索你想要下载的插件
